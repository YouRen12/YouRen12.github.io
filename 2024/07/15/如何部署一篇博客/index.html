Ⅰ.如何部署一篇博客

一、git

首先需要 git
没有 git 的话可以去这里下载

1、官方下载地址： Git - Downloading Package (git-scm.com)

二、Node.js

然后需要 Node.js
没有的话去下面这里下载

https://nodejs.org/zh-cn/
cmd 中输入如下指令：node -v，出现 node.js 版本信息，说明安装成功

安装 Hexo 框架

1.然后我们开始安装 Hexo 框架
首先是通过 cmd 安装 cnpm

npm install -g cnpm –registry==https://registry.npm.taobao.org
安装好以后，通过 cmd 输入 cnpm -v, 出现 cnpm 版本信息说明安装成功

2、安装 hexo
npm install -g cnpm –registry==https://registry.npm.taobao.org

安装好以后，可在 cmd 中输入 hexo -v, 出现 hexo 版本信息说明安装成功

三、然后新建一个文件夹 名称随意 建议叫 hexo

在该文件夹下右键选择 Open Git Bash here

在 git 窗口，输入：hexo init，进行项目的初始化 等待一段时间 出现感叹号即表示初始化完成
​ 初始化完成后，在存放项目的文件夹中会自动出现配置文件
在 git 窗口，输入:hexo s,开启本地预览服务，打开浏览器访问 http://localhost:4000 即可看到博客内容

四、至此，博客网站已经初步配置成功

部署完成以后，主题是默认主题，接下来我们需要更改主题样式

Ⅱ、样式修改

1.可以通过 Themes 获取主题样式，也可以直接在 github 上搜索 hexo 主题，来获取别人开源的样式

Themes 官网

https://hexo.io/themes/

2.我这里演示从 github 上拉取开源样式，因为 github 的样式好看

github 地址

https://github.com/ohoh3000/hexo-theme-mokusei
进入链接以后,点击右上角 code,获取地址

https://github.com/ohoh3000/hexo-theme-mokusei.git

在项目文件夹下，使用 Open Git Bash here 打开 git 窗口，输入以下指令（git clone 主题 http 链接 themes/主题名称

git clone https://github.com/ohoh3000/hexo-theme-mokusei.git themes/mokusei
然后耐心等待主题下载

下载完毕以后,在在 themse 文件夹下会出现名为：hexo-theme-chan 的主题文件夹

然后打开 vscode,更改博客配置文件_config.yml, theme 选项值设为 mokusei

_config._config.yml

- theme: other themes
+ theme: mokusei

关于代码高亮 (更改博客配置文件_config.yml 为以下推荐配置)

_config._config.yml

highlight:
  enable: true
  line_number: false
  auto_detect: false
  tab_replace: ''
  wrap: false
  hljs: true
prismjs:
  enable: false
  preprocess: true
  line_number: true
  tab_replace: ''

三、然后回到 git,使用以下命令更新主题

hexo clean //执行此命令后继续下一条
hexo g //生成博客目录
hexo s //本地预览

至此博客样式更换成功

Ⅲ、博客如何使用

我们可以查阅样式的 api 文档,然后了解使用方法

https://hexo.io/docs/writing.html

1.比如说我们可以通过 git 创建一个帖子

$ hexo new [布局] <标题>
post是默认的layout，但您可以提供自己的布局。您可以通过编辑default_layout中的设置来更改默认布局_config.yml。

创建好博客以后,我们通过文件夹,比如说这个主题的书写文件所在位置

Hexo—>source—>_post—->.md 文件

2.进入这个页面以后,我们可以通过 markdown 的用法.来书写界面

Markdown官网
https://www.runoob.com/markdown/md-paragraph.html

crtl+s 保存.然后刷新页面即可查看书写内容
